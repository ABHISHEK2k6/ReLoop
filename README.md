This is a prototype of the ReWaste Waste Exchange Platform, connecting waste generators with recycling buyers.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Giver Portal**: List waste for collection
- **Buyer Portal**: Post material demands
- Real-time feed of listings and demands
- Geolocation map interface
- Material type filtering

## Tech Stack

- Next.js 14 (App Router)
- React 18
- CSS-in-JS with custom properties

## Note

This is a prototype version using mock data. Backend integration and database will be added in future iterations.
